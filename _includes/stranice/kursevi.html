Izaberi svoje buduće zanimanje

Web dizajner je osoba koja kodira web stranice, implementirajući dizajn. Glavna aktivnost web dizajnera je izrada stranica i prevođenje dizajna (obično PSD fajlova) u kod.

Front-end developer je specijalista za izradu svega vidljivog na webu. Frontend developer je sve traženije IT zanimanje, u koje se relativno lako ulazi (kasnije stvari postaju složene).

PHP Developer je specijalista za velike serverske aplikacije, poput poznatih kupoprodajnih sajtova. PHP je najpopularniji backend jezik u Srbiji i zanimanje je veoma traženo.

Node.js Developer je relativno novo back-end zanimanje, orjentisano na razvoj web aplikacija i API servisa. Node.js trenutno ima najveći ekosistem od svih jezika ikad.

Web developer je poput lekara opšte prakse - poznaje glavne web tehnologije, ali ni u jednoj nije specijalista. Pozicija web developera karakteristična je za manje firme.

3D Web developer je retko i visoko specijalizovano zanimanje. Oglasa za posao nema puno, ali firme u Srbiji često i po nekoliko meseci traže Three.js programera.

Javascript developer je specijalizovani programer koji odlično poznaje sam jezik i glavne biblioteke. Veoma traženo zanimanje, jer je pomoću JS-a moguće praviti softver za sve platforme i operativne sisteme.

Full Stack Web Developer je kompletan developer koji dobro poznaje sve aspekte izrade profesionalnih web aplikacija, od baza podataka do dizajna. Iako vlada svim tehnologijama u procesu, obično ima svoje polje specijalizacije.

Programer u povoju nije zanimanje koje postoji u oglasima za posao, ali je veoma dobar start za svakog ko je rešio da pođe putem razvoja softvera.
